## Architecture

### New Cloud Functions (added to `functions/index.js`)

| Function | Trigger | Purpose |
|---|---|---|
| `generateAILeads` | HTTP POST | Gemini generates 5 fictional/plausible leads for a niche + location. Saves to `leads` collection. |
| `generateOutreachScript` | HTTP POST | Gemini writes a collapsible sales script for a given lead. |
| `sendLeadAlert` | HTTP POST | Sends lead notifications to Discord webhook, Telegram bot, or Twilio WhatsApp. No auth gate. Two modes: `single` (one lead) or `batch` (generation summary). |
| `processLeadStreams` | Scheduled (Mon 8am SAST) | Reads `leadStreamSubscriptions`, generates leads per client, calls `sendLeadAlert` for each channel. |

**Existing functions reused:**
- `generateOutreachEmail` (Gemini) â€” already handles email generation, reused for Email tab
- `getAI()` helper â€” same pattern used for all new Gemini calls

### Firestore Collections

**`leads`** (existing) â€” AI-generated leads saved here, tagged with `source: 'ai_hunter'` and optional `clientId` field for Lead Stream clients.

**`leadStreamSubscriptions`** (new):
```
{
  clientName: string,
  clientEmail: string,
  niche: string,
  location: string,
  companySize: string,
  frequency: 'weekly',
  whatsapp: string | null,
  discordWebhook: string | null,
  telegramChatId: string | null,
  active: boolean,
  createdAt: Timestamp,
  lastRunAt: Timestamp | null
}
```

### Security

- `sendLeadAlert` â€” no auth gate. Webhook URLs stored in Firebase secrets (`DISCORD_WEBHOOK_URL`, `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`, `TWILIO_ACCOUNT_SID`, `TWILIO_AUTH_TOKEN`, `TWILIO_WHATSAPP_FROM`). Never exposed to client.
- `generateAILeads` / `generateOutreachScript` â€” open CORS (same as existing functions), rate limiting handled by Gemini quota.
- `processLeadStreams` â€” scheduled, no HTTP exposure.
- All user inputs (niche, location, lead fields) are passed as prompt context only â€” never executed as code.

## Lead Stream Service

### How It Works
1. Client signs up for Lead Stream (add-on on Pricing page, R299/month)
2. Drift Studio creates a `leadStreamSubscriptions` doc for them
3. Every Monday 8am SAST, `processLeadStreams` runs
4. For each active subscription: generates 5 leads â†’ saves to `leads` collection â†’ sends formatted message to client's WhatsApp/Discord/Telegram
5. Client receives leads directly in their preferred app, no login required
